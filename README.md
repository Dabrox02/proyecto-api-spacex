# Proyecto API SpaceX

Este repositorio contiene ejemplos de consultas a la API pública de SpaceX. La API de SpaceX proporciona acceso a datos relacionados con las misiones espaciales, cohetes, naves espaciales y otros aspectos del programa espacial de SpaceX.

## Introducción

Este repositorio tiene como objetivo contribuir con consultas a la API v4 de SpaceX en los modulos de:

- Capsules
- Cores
- Launches
- Rockets
- Ships

## Requisitos

Asegúrate de tener las siguientes herramientas y recursos instalados antes de ejecutar las consultas:

-  [ThunderClient](https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client) para VS Code o [Postman](https://www.postman.com/downloads/).
